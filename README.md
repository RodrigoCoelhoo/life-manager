> [!NOTE]  
> **Life Manager** is an all-in-one personal management app designed to help users organize and track the most important areas of their life: **Finances, Nutrition, and Training**.  
>  
> The goal is to provide clear insights through automation and dashboards, while keeping daily input simple and flexible.

# Features Overview

### üí∞ Finances

Gain full control over your personal finances.

#### Wallets
- Create multiple wallets (e.g. Cash, Bank Account)
- Each wallet maintains a complete history of:
  - Transactions
  - Transfers between wallets

#### Automated Bills
- Create recurring bills (monthly, yearly, etc.)
- Bills are automatically applied to the corresponding wallet

#### Monthly Financial Dashboard
- Overview of income, expenses, and balances
- Summary of all wallets and recurring bills for the month

---

### ü•ó Nutrition

Track nutrition with precision.

#### Ingredients & Brands
- Create ingredients (e.g. Rice, Chicken, Milk)
- For each ingredient, define multiple brands
- Each brand can have its own nutritional values

#### Meals
- Build meals using ingredient brands and specific quantities
- Nutritional values are automatically calculated based on amounts

#### Recipes
- Create reusable recipes to speed up meal creation
- Recipes automatically populate meals with predefined ingredients and quantities

#### Weekly Nutrition Dashboard
- Clear overview of weekly:
  - Calories
  - Protein
  - Carbohydrates
  - Fats
  - Other macros
